# Django-React-Api-GameStore
Api feita em django que é acessada pelo fronted construido em react

backend:
para rodar basta instalar as dependencias em requirements.txt com os comandos:
cd ecommerce_api 
pip install -r requirements.txt
inicie a api com:
python manage.py runserver

frontend:
instale as dependências com:
npm install
inicie o projeto com:
npm start
